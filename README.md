# SpeakSmart (SLW Academy)

SpeakSmart is a free browser-based toolkit for public speaking students and educators.
It includes six tools in one app:

1. Speech Title Generator
2. Speech Outline Generator
3. Impromptu Topic Generator
4. Word of the Day
5. Joke of the Day
6. Quote of the Day

## Tech Stack

- HTML
- CSS
- JavaScript
- No frameworks
- No backend
- No API calls

## Features Implemented

- Age group selector (required before generating content):
  - Ages 10–13 (Middle School)
  - Ages 14–18 (High School)
  - Ages 18+ (College & Professionals)
- Content adapted by age group where required
- No-repeat generation logic for:
  - Impromptu topics (50 per age group)
  - Words of the day (50 per age group)
  - Jokes of the day (50 per age group)
  - Quotes (20 per theme)
- Automatic reshuffle only after all items are used
- SLW brand colors and mobile-responsive UI
- Updated SLW Academy logo in header
- Footer:
  - Built by SpeakLeadWin · SLW Academy Digital Intern 2026 · speakleadwinacademy.com

## Run Locally

1. Download or clone this repository.
2. Open `index.html` in any browser.
3. Select an age group first.
4. Use tabs and generate content.

## File Structure

- `index.html` — app markup and tab layout
- `styles.css` — branding and responsive styling
- `script.js` — app logic and no-repeat generation engine
- `assets/slw-academy-logo.jpg` — SLW logo shown in header
